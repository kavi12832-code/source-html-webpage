# My Portfolio Website

A simple, multi-page portfolio website built with HTML, CSS, and JavaScript — created as a learning project to practice web fundamentals, Git version control, and deployment via GitHub Pages.

## Live Demo

🔗 [View Live Site](https://kavi12832-code.github.io/source-html-webpage/)

## Pages

- **Home** (`index.html`) — Landing page with a welcome message and call-to-action
- **About** (`about.html`) — Brief introduction about me
- **Projects** (`projects.html`) — Showcase of projects I've built
- **Contact** (`contact.html`) — A simple contact form (demo only, no backend)

## Tech Stack

- HTML5
- CSS3 (custom styling, no frameworks)
- Vanilla JavaScript (form interaction)
